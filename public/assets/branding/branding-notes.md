# Virtura Brand Notes

## Core Feel
Modern, futuristic, premium, trustworthy, tech-forward, clean.

## Core Colors
- Primary blue: #2F6FE4
- Dark slate text: #374151
- Light neutral background: #F3F4F6

## Style
- Slightly futuristic
- Clean geometry
- Eye-catching but professional
- Not overly playful
- Not generic corporate

## Usage
- Virtura Group blue is the main accent for Presence
- Use sub-brand colors only for supporting references
- Avoid overly bright neon overload
